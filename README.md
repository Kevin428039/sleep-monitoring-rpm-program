# Sleep Monitoring RPM Program

Remote monitoring program for sleep and neurological patients.

## Services Covered
- PSG follow-up
- CPAP compliance monitoring
- Oral appliance therapy monitoring
- EEG monitoring

## Program Workflow

Patient Diagnosis  
↓  
Therapy Initiated  
↓  
Remote Monitoring  
↓  
Alert Detection  
↓  
Technologist Review  
↓  
Physician Intervention

## Monitoring Metrics

| Therapy | Key Metrics |
|--------|-------------|
| CPAP | Usage hours, leak rate, residual AHI |
| Oral Appliance | Snoring reduction, sleepiness score |
| EEG | seizure events, abnormal spikes |
| PSG | treatment follow-up |

## Goals

- Improve CPAP compliance
- Detect therapy failures early
- Track neurological sleep disorders
- Improve patient outcomes
